# Aula 1.1 – Configuração e Janela OpenGL

Primeira entrega prática do curso, versão simplificada focada no Windows. Implementa o bootstrap essencial com GLFW + GLAD para abrir uma janela OpenGL 3.3 Core.

## Como executar
```bash
build.bat    # gera solução e compila
run.bat      # executa a aplicação
```

Scripts reutilizam dependências do projeto `ultraMini`.

## Estrutura
```
Aula 1.1/
├── build.bat
├── premake5.lua
├── run.bat
└── src/
    └── main.cpp
```
